# Clinic API

A RESTful API built with ASP.NET Core (.NET 8) for managing a clinic system.

---

## 🚀 Features
- Manage Doctors, Patients, and Appointments
- Full CRUD operations
- Entity Framework Core integration
- SQL Server database
- Swagger API documentation

---

## 🛠 Tech Stack
- Backend: ASP.NET Core (.NET 8)
- Database: SQL Server
- ORM: Entity Framework Core
- Tools: Swagger, REST APIs

---

### Run the API

```bash
dotnet restore
dotnet run
